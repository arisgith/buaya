Selamat datang di buaya.KLAS.or.id!
buaya.KLAS.or.id adalah cermin komunitas yang menyediakan repository perangkat lunak kode sumber terbuka open source software, pustaka library, dokumentasi teknis dan beberapa material lain.

buaya.KLAS.or.id adalah persembahan Kelompok Linux Arek Surabaya untuk komunitas, pengguna open source di semua tempat.
